Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tab <b> (não semântica).

Nesta frase temos um termo destacado usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo ítalico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também textos marcados usando a tag <mark>.

E no outro parágrafo temos outro texto marcado.

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase que ele foi adicionado depois.

Existe também o sublinhado com a tag <u> (não sêmantica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3.

Texto Subscrito

Para inserir coisas do tipo H20.
